## Start guide
From the root folder launch build script 
~~~
  ./build.sh
~~~
After successful build start docker containers
~~~
  docker-compose up
~~~
## Requirements
- Open JDK 17
- Apache Maven 3.8
- npm 9.5
- Angular CLI 16
- Node 18

**Warning:** Functionality on other versions is not guaranteed! ⚠️
